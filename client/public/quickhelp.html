🚆 Quick Help

General Controls

Left Click
Select element or place new element (draw mode)
Ctrl + Drag
Pan (move) the canvas
Mouse Wheel
Zoom in / out
ESC
Switch back to cursor mode

Editing

Click
Select element
Drag
Move selected element
Delete / Backspace
Delete selected element

Rotation

R
Rotate selected element
R (Draw Mode)
Rotate preview (ghost element)

Undo / Redo

Ctrl + Z
Undo last action
Ctrl + Y
Redo last undone action
Ctrl + Shift + Z
Redo (alternative shortcut)

Modes

Cursor Mode
Select and move elements
Draw Mode
Place new elements on the grid
Tip: Use zoom + pan for precise placement. Undo/Redo helps experimenting safely.
